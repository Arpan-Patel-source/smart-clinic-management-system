# User Stories

## 👤 Patient User Stories

- **As a Patient**, I want to be able to register an account in the system so that I can securely manage my personal and medical details.

- **As a Patient**, I want to view a list of available doctors and their specialties so that I can choose the right doctor for my medical needs.

- **As a Patient**, I want to be able to book, reschedule, or cancel an appointment so that I can manage my medical visits conveniently.

- **As a Patient**, I want to receive confirmation for my booked appointments so that I do not miss my scheduled visits.

- **As a Patient**, I want to search for a doctor by name or specialty so that I can find the right medical professional quickly.

---

## 🩺 Doctor User Stories

- **As a Doctor**, I want to view my daily and weekly appointment schedules so that I can prepare for my patient consultations.

- **As a Doctor**, I want to update my available time slots so that patients can only book me when I am available.

- **As a Doctor**, I want to log in securely using my email and password so that my account and patient data are protected.

- **As a Doctor**, I want to add consultation notes and prescriptions after an appointment so that I have a medical record for the patient's future visits.

- **As a Doctor**, I want to view my patient's basic information before a consultation so that I can provide accurate medical advice.

---

## 🔧 Admin User Stories

- **As an Admin**, I want to add, update, and delete doctor profiles so that the clinic's doctor database remains accurate and up-to-date.

- **As an Admin**, I want to add and manage patient records so that patient information is always current and accessible.

- **As an Admin**, I want to view all appointments across all doctors so that I can monitor clinic operations and resolve scheduling conflicts.

- **As an Admin**, I want to run reports showing appointment counts per doctor so that I can make informed clinic management decisions.

- **As an Admin**, I want to manage the system securely with admin credentials so that unauthorized users cannot access sensitive data.
